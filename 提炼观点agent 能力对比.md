## 各模型详细分析

### Claude Sonnet（首选日常）

财经文章提炼最强的地方在于**结构化输出质量高、不添油加醋**。公众号财经文章普遍有"标题党"、情绪化表达、夹带私货的问题，Claude 的一个特点是提炼出来的核心观点更克制、更准确，不会被文章本身的情绪带跑。

另外你已经有 Claude API，成本边际为零，最适合日常高频使用。

**适合**：单篇精读、输出结构化摘要、识别文章逻辑漏洞。

---

### Gemini 2.5 Pro（深度分析首选）

100 万 token 的上下文是核心优势。你每天几十篇文章，完全可以**把一天的所有文章一次性塞进去**，让它做跨文章的对比分析——哪些观点是主流、哪些是少数派、哪些观点相互矛盾。这种"全局视角"分析是其他模型做不到的。

财经推理能力和 Claude 基本持平，在数学/量化逻辑上略强。

**适合**：每日文章批量汇总、跨文章观点聚合、识别市场情绪趋势。

---

### DeepSeek V3（批量处理首选）

成本是 Claude 和 GPT-4o 的 1/10 甚至更低，中文能力是国产模型里最强的，速度也快。每天几十篇文章如果全走 API，用 DeepSeek 可以把成本压到可以忽略不计的程度。

观点提炼质量比 Claude 略差，偶尔会"顺着"文章逻辑走，批判性稍弱。

**适合**：初筛、批量生成摘要、成本敏感场景。

---

### GPT-4o（不推荐作主力）

能力不差，但在这个场景里没有明显优势。你已经有 ChatGPT Plus 通过 sub2api 提供给朋友用了，拿来做文章分析是大材小用，而且 API 价格比 Claude 贵，中文细腻程度不如 Claude 和 DeepSeek。

## 推荐的工作流

结合你的现有基础设施：

```
WeWe RSS 拉取文章 URL
    ↓
Python 脚本抓正文入库（SQLite）
    ↓
┌─────────────────────────────────┐
│  DeepSeek V3：批量初筛          │  → 过滤掉无价值文章
│  每篇输出：100字摘要 + 重要度评分 │
└─────────────────────────────────┘
    ↓（重要度 > 7分的文章）
┌─────────────────────────────────┐
│  Claude Sonnet：精读提炼        │  → 核心观点 + 逻辑验证
│  结构化输出：论点/论据/结论/风险  │
└─────────────────────────────────┘
    ↓（每日收盘后）
┌─────────────────────────────────┐
│  Gemini 2.5 Pro：每日汇总       │  → 当日市场观点全景图
│  输入：当日所有精选文章           │
└─────────────────────────────────┘
```

这套流程的日成本：几十篇文章全走下来大概不到 **1 块钱人民币**（DeepSeek 初筛极便宜，Claude 只处理精选的，Gemini 2.5 Pro 每天一次汇总）。
